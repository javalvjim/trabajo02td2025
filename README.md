# Introducción

Este repositorio **"trabajo02td2025"** contiene el segundo trabajo de la asignatura Teoría de la Decisión, en el 
que se resuelve el problema *Elección de la mejor sede para hacer los juegos olímpicos*, mediante los cuatro métodos
vistos en clase: AHP, ELECTRE, PROMETHEE, ARROW-RAYMOND.

Se persiguen varios objetivos con este repositorio:

1. Encontrar fácilmente el código actualizado del primer trabajo de la asignatura.

2. Conocer e interactuar con el proceso del control de versiones de scripts de un código con el sistema Git-Github.
